## What I'm building

**[Bookkeeping Admin](https://github.com/fotisanpetprojects/bookkeeping-admin)**  
A local-first bookkeeping tool for independent professionals and small service businesses in the Netherlands. It covers billing profiles, invoices, expenses, receipts, and quarterly BTW/VAT summaries, hosted locally. 

**[Tulip Field Generator](https://github.com/fotisanpetprojects/tulip-field)**  
A Chrome extension that turns the visible layout of a webpage into a stylised Dutch tulip field. It detects page blocks, clusters layout regions, and renders procedural fields over the page.


## How I work

- Start with the user problem, not the tool.
- Scope the smallest useful version first.
- Use AI as a delivery partner, but keep product judgment in the loop.
- Make tradeoffs explicit: privacy, complexity, cost, speed, maintainability.
- Document what changed, why it matters and what should come next.
